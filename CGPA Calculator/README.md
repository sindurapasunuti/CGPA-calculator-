CGPA Calculator

Description:
A simple web-based CGPA Calculator developed using HTML, CSS, and JavaScript.

Features:
- Enter number of subjects
- Enter grade points
- Calculate CGPA instantly
- Simple user interface

Technologies Used:
- HTML
- CSS
- JavaScript
- Git
- GitHub

How to Run:
1. Open index.html in a web browser.
2. Enter the number of subjects.
3. Enter grade points.
4. Click Calculate CGPA.
